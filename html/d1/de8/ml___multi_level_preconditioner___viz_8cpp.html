visualize aggregates and compute some statistics on the aggregates, like their diameter and number of nodes (for all levels).

This function performs a post-processing of data. To use this function, the user should do the following:

  • before calling ML_Gen_MGHierarchy_UsingAggregation, he/she has to initialize a ML_Aggregate_Viz_Stats list (internally stored in the ML_Aggregate object). This is done as "ML_Aggregate_Viz_Stats_Create( ml_ag, MaxLevels )".
  • then, function `ML_Aggregate_CoarsenMETIS' will recognize that an ML_Aggregate_Viz_Stats object has been created, and she will put a copy of the graph decomposition into the ML_Aggre_Info object of that level (note that levels moves from 0 <fine> to # levels )
  • after a call to this function, he/she need to free memory, using "ML_Aggregate_Viz_Stats_Destroy"
