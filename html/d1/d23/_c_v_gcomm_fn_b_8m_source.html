1 %CVGcommFn - type for user provided communication function (BBDPre) for backward problems.
2 %
3 % The function GCOMFUNB must be defined either as
4 % FUNCTION FLAG = GCOMFUNB(T, Y, YB)
5 % or as
6 % FUNCTION [FLAG, NEW_DATA] = GCOMFUNB(T, Y, YB, DATA)
7 % depending on whether a user data structure DATA was specified in
8 % CVodeInit.
9 %
10 % The function GCOMFUNB must set FLAG=0 if successful, FLAG<0 if an
11 % unrecoverable failure occurred, or FLAG>0 if a recoverable error
12 % occurred.
13 %
14 % See also CVGlocalFnB, CVodeSetOptions
15 %
16 % NOTES:
17 % GCOMFUNB is specified through the GcommFn property in CVodeSetOptions
18 % and is used only if the property PrecModule is set to 'BBDPre'.
19 %
20 % Each call to GCOMFUNB is preceded by a call to the RHS function
21 % ODEFUNB with the same arguments T, Y, and YB. Thus GCOMFUNB can
22 % omit any communication done by ODEFUNB if relevant to the evaluation
23 % of G by GLOCFUNB. If all necessary communication was done by ODEFUNB,
24 % GCOMFUNB need not be provided.
