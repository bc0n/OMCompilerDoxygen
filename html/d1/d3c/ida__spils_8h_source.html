46 /*
47  * -----------------------------------------------------------------
48  * Type : IDASpilsPrecSetupFn
49  * -----------------------------------------------------------------
50  * The optional user-supplied functions PrecSetup and PrecSolve
51  * together must define the left preconditoner matrix P
52  * approximating the system Jacobian matrix
53  * J = dF/dy + c_j*dF/dy'
54  * (where the DAE system is F(t,y,y') = 0), and solve the linear
55  * systems P z = r. PrecSetup is to do any necessary setup
56  * operations, and PrecSolve is to compute the solution of
57  * P z = r.
58  *
59  * The preconditioner setup function PrecSetup is to evaluate and
60  * preprocess any Jacobian-related data needed by the
61  * preconditioner solve function PrecSolve. This might include
62  * forming a crude approximate Jacobian, and performing an LU
63  * factorization on it. This function will not be called in
64  * advance of every call to PrecSolve, but instead will be called
65  * only as often as necessary to achieve convergence within the
66  * Newton iteration. If the PrecSolve function needs no
67  * preparation, the PrecSetup function can be NULL.
68  *
69  * Each call to the PrecSetup function is preceded by a call to
70  * the system function res with the same (t,y,y') arguments.
71  * Thus the PrecSetup function can use any auxiliary data that is
72  * computed and saved by the res function and made accessible
73  * to PrecSetup.
74  *
75  * A preconditioner setup function PrecSetup must have the
76  * prototype given below. Its parameters are as follows:
77  *
78  * tt is the current value of the independent variable t.
79  *
80  * yy is the current value of the dependent variable vector,
81  * namely the predicted value of y(t).
82  *
83  * yp is the current value of the derivative vector y',
84  * namely the predicted value of y'(t).
85  *
86  * rr is the current value of the residual vector F(t,y,y').
87  *
88  * c_j is the scalar in the system Jacobian, proportional to 1/hh.
89  *
90  * user_data is a pointer to user data, the same as the user_data
91  * parameter passed to IDASetUserData.
92  *
93  * tmp1, tmp2, tmp3 are pointers to vectors of type N_Vector
94  * which can be used by an IDASpilsPrecSetupFn routine
95  * as temporary storage or work space.
96  *
97  * NOTE: If the user's preconditioner needs other quantities,
98  * they are accessible as follows: hcur (the current stepsize)
99  * and ewt (the error weight vector) are accessible through
100  * IDAGetCurrentStep and IDAGetErrWeights, respectively (see
101  * ida.h). The unit roundoff is available as
102  * UNIT_ROUNDOFF defined in sundials_types.h
103  *
104  * The IDASpilsPrecSetupFn should return
105  * 0 if successful,
106  * a positive int if a recoverable error occurred, or
107  * a negative int if a nonrecoverable error occurred.
108  * In the case of a recoverable error return, the integrator will
109  * attempt to recover by reducing the stepsize (which changes cj).
110  * -----------------------------------------------------------------
111  */
112 
113 typedef int (*IDASpilsPrecSetupFn)(realtype tt,
114  N_Vector yy, N_Vector yp, N_Vector rr,
115  realtype c_j, void *user_data,
116  N_Vector tmp1, N_Vector tmp2,
117  N_Vector tmp3);

119 /*
120  * -----------------------------------------------------------------
121  * Type : IDASpilsPrecSolveFn
122  * -----------------------------------------------------------------
123  * The optional user-supplied function PrecSolve must compute a
124  * solution to the linear system P z = r, where P is the left
125  * preconditioner defined by the user. If no preconditioning
126  * is desired, pass NULL for PrecSolve to IDASp*.
127  *
128  * A preconditioner solve function PrecSolve must have the
129  * prototype given below. Its parameters are as follows:
130  *
131  * tt is the current value of the independent variable t.
132  *
133  * yy is the current value of the dependent variable vector y.
134  *
135  * yp is the current value of the derivative vector y'.
136  *
137  * rr is the current value of the residual vector F(t,y,y').
138  *
139  * rvec is the input right-hand side vector r.
140  *
141  * zvec is the computed solution vector z.
142  *
143  * c_j is the scalar in the system Jacobian, proportional to 1/hh.
144  *
145  * delta is an input tolerance for use by PrecSolve if it uses an
146  * iterative method in its solution. In that case, the
147  * the residual vector r - P z of the system should be
148  * made less than delta in weighted L2 norm, i.e.,
149  * sqrt [ Sum (Res[i]*ewt[i])^2 ] < delta .
150  * Note: the error weight vector ewt can be obtained
151  * through a call to the routine IDAGetErrWeights.
152  *
153  * user_data is a pointer to user data, the same as the user_data
154  * parameter passed to IDASetUserData.
155  *
156  * tmp is an N_Vector which can be used by the PrecSolve
157  * routine as temporary storage or work space.
158  *
159  * The IDASpilsPrecSolveFn should return
160  * 0 if successful,
161  * a positive int if a recoverable error occurred, or
162  * a negative int if a nonrecoverable error occurred.
163  * Following a recoverable error, the integrator will attempt to
164  * recover by updating the preconditioner and/or reducing the
165  * stepsize.
166  * -----------------------------------------------------------------
167  */
168

175 /*
176  * -----------------------------------------------------------------
177  * Type : IDASpilsJacTimesVecFn
178  * -----------------------------------------------------------------
179  * The user-supplied function jtimes is to generate the product
180  * J*v for given v, where J is the Jacobian matrix
181  * J = dF/dy + c_j*dF/dy'
182  * or an approximation to it, and v is a given vector.
183  * It should return 0 if successful and a nonzero int otherwise.
184  *
185  * A function jtimes must have the prototype given below. Its
186  * parameters are as follows:
187  *
188  * tt is the current value of the independent variable.
189  *
190  * yy is the current value of the dependent variable vector,
191  * namely the predicted value of y(t).
192  *
193  * yp is the current value of the derivative vector y',
194  * namely the predicted value of y'(t).
195  *
196  * rr is the current value of the residual vector F(t,y,y').
197  *
198  * v is the N_Vector to be multiplied by J.
199  *
200  * Jv is the output N_Vector containing J*v.
201  *
202  * c_j is the scalar in the system Jacobian, proportional
203  * to 1/hh.
204  *
205  * user_data is a pointer to user data, the same as the
206  * pointer passed to IDASetUserData.
207  *
208  * tmp1, tmp2 are two N_Vectors which can be used by Jtimes for
209  * work space.
210  * -----------------------------------------------------------------
211  */

220 /*

222  * Optional inputs to the IDASPILS linear solver
223  * -----------------------------------------------------------------
224  *
225  * IDASpilsSetPreconditioner specifies the PrecSetup and PrecSolve
226  * functions.
227  * Default is NULL for both arguments.
228  * IDASpilsSetJacTimesVecFn specifies the jtimes function.
229  * Default is to use an internal finite difference
230  * approximation routine.
231  * IDASpilsSetGSType specifies the type of Gram-Schmidt
232  * orthogonalization to be used. This must be one of
233  * the two enumeration constants MODIFIED_GS or
234  * CLASSICAL_GS defined in iterativ.h. These correspond
235  * to using modified Gram-Schmidt and classical
236  * Gram-Schmidt, respectively.
237  * Default value is MODIFIED_GS.
238  * Only for IDASPGMR.
239  * IDASpilsSetMaxRestarts specifies the maximum number of restarts
240  * to be used in the GMRES algorithm. maxrs must be a
241  * non-negative integer. Pass 0 to specify no restarts.
242  * Default is 5.
243  * Only for IDASPGMR.
244  * IDASpbcgSetMaxl specifies the maximum Krylov subspace size.
245  * Default is 5.
246  * Only for IDASPBCG and IDASPTFQMR.
247  * IDASpilsSetEpsLin specifies the factor in the linear iteration
248  * convergence test constant.
249  * Default is 0.05
250  * IDASpilsSetIncrementFactor specifies a factor in the increments
251  * to yy used in the difference quotient approximations
252  * to matrix-vector products Jv.
253  * Default is 1.0
254  *
255  * The return value of IDASpilsSet* is one of:
256  * IDASPILS_SUCCESS if successful
257  * IDASPILS_MEM_NULL if the ida memory was NULL
258  * IDASPILS_LMEM_NULL if the linear solver memory was NULL
259  * -----------------------------------------------------------------
