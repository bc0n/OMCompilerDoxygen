1 %CVPrecSetupFnB - type for user provided preconditioner setup function for backward problems.
2 %
3 % The user-supplied preconditioner setup function PSETFUN and
4 % the user-supplied preconditioner solve function PSOLFUN
5 % together must define left and right preconditoner matrices
6 % P1 and P2 (either of which may be trivial), such that the
7 % product P1*P2 is an approximation to the Newton matrix
8 % M = I - gamma*J. Here J is the system Jacobian J = df/dy,
9 % and gamma is a scalar proportional to the integration step
10 % size h. The solution of systems P z = r, with P = P1 or P2,
11 % is to be carried out by the PrecSolve function, and PSETFUN
12 % is to do any necessary setup operations.
13 %
14 % The user-supplied preconditioner setup function PSETFUN
15 % is to evaluate and preprocess any Jacobian-related data
16 % needed by the preconditioner solve function PSOLFUN.
17 % This might include forming a crude approximate Jacobian,
18 % and performing an LU factorization on the resulting
19 % approximation to M. This function will not be called in
20 % advance of every call to PSOLFUN, but instead will be called
21 % only as often as necessary to achieve convergence within the
22 % Newton iteration. If the PSOLFUN function needs no
23 % preparation, the PSETFUN function need not be provided.
24 %
25 % For greater efficiency, the PSETFUN function may save
26 % Jacobian-related data and reuse it, rather than generating it
27 % from scratch. In this case, it should use the input flag JOK
28 % to decide whether to recompute the data, and set the output
29 % flag JCUR accordingly.
30 %
31 % Each call to the PSETFUN function is preceded by a call to
32 % ODEFUN with the same (t,y) arguments. Thus the PSETFUN
33 % function can use any auxiliary data that is computed and
34 % saved by the ODEFUN function and made accessible to PSETFUN.
35 %
36 %
37 % The function PSETFUNB must be defined either as
38 % FUNCTION [JCURB, FLAG] = PSETFUNB(T,Y,YB,FYB,JOK,GAMMAB)
39 % or as
40 % FUNCTION [JCURB, FLAG, NEW_DATA] = PSETFUNB(T,Y,YB,FYB,JOK,GAMMAB,DATA)
41 % depending on whether a user data structure DATA was specified in
42 % CVodeInit. In either case, it must return the flags JCURB and FLAG.
43 %
44 % See also CVPrecSolveFnB, CVodeSetOptions
45 %
46 % NOTE: PSETFUNB is specified through the property PrecSetupFn to
47 % CVodeSetOptions and is used only if the property LinearSolver was
48 % set to 'GMRES', 'BiCGStab', or 'TFQMR' and if the property PrecType
49 % is not 'None'.
