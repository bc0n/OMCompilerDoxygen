1 function [varargout] = IDASolve(tout,itask)
2 %IDASolve integrates the DAE.
3 %
4 % Usage: [STATUS, T, Y] = IDASolve ( TOUT, ITASK )
5 % [STATUS, T, Y, YQ] = IDASolve (TOUT, ITASK )
6 % [STATUS, T, Y, YS] = IDASolve ( TOUT, ITASK )
7 % [STATUS, T, Y, YQ, YS] = IDASolve ( TOUT, ITASK )
8 %
9 % If ITASK is 'Normal', then the solver integrates from its current internal
10 % T value to a point at or beyond TOUT, then interpolates to T = TOUT and returns
11 % Y(TOUT). If ITASK is 'OneStep', then the solver takes one internal time step
12 % and returns in Y the solution at the new internal time. In this case, TOUT
13 % is used only during the first call to IDASolve to determine the direction of
14 % integration and the rough scale of the problem. In either case, the time
15 % reached by the solver is returned in T.
16 %
17 % If quadratures were computed (see IDAQuadInit), IDASolve will return their
18 % values at T in the vector YQ.
19 %
20 % If sensitivity calculations were enabled (see IDASensInit), IDASolve will
21 % return their values at T in the matrix YS. Each row in the matrix YS
22 % represents the sensitivity vector with respect to one of the problem parameters.
23 %
24 % In ITASK =' Normal' mode, to obtain solutions at specific times T0,T1,...,TFINAL
25 % (all increasing or all decreasing) use TOUT = [T0 T1 ... TFINAL]. In this case
26 % the output arguments Y and YQ are matrices, each column representing the solution
27 % vector at the corresponding time returned in the vector T. If computed, the
28 % sensitivities are eturned in the 3-dimensional array YS, with YS(:,:,I) representing
29 % the sensitivity vectors at the time T(I).
30 %
31 % On return, STATUS is one of the following:
32 % 0: IDASolve succeeded and no roots were found.
33 % 1: IDASolve succeded and returned at tstop.
34 % 2: IDASolve succeeded, and found one or more roots.
35 % -1: An error occurred (see printed message).
36 %
37 % See also IDASetOptions, IDAGetStats
