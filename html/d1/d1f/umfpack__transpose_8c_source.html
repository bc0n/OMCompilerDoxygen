11 /*
12  User callable. Computes a permuted transpose, R = (A (P,Q))' in MATLAB
13  notation. See umfpack_transpose.h for details. A and R can be rectangular.
14  The matrix A may be singular.
15  The complex version can do transpose (') or array transpose (.').
16 
17  Dynamic memory usage: A single call to UMF_malloc is made, for a workspace
18  of size max (n_row,n_col,1) * sizeof(Int). This is then free'd on return,
19  via UMF_free.
20 */
