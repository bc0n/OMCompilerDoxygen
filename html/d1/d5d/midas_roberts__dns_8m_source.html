2 %midasRoberts_dns - IDAS example problem (serial, dense)
3 % The following is a simple example problem, with the coding
4 % needed for its solution by IDAS. The problem is from
5 % chemical kinetics, and consists of the following three rate
6 % equations:
7 % dy1/dt = -.04*y1 + 1.e4*y2*y3
8 % dy2/dt = .04*y1 - 1.e4*y2*y3 - 3.e7*(y2)^2
9 % 1 = y1 + y2 + y3
10 % on the interval from t = 0.0 to t = 4.e10, with initial
11 % conditions: y1 = 1.0, y2 = y3 = 0. The problem is stiff.
12 % While integrating the system, we also use the rootfinding
13 % feature to find the points at which y1 = 1e-4 or at which
14 % y3 = 0.01.
