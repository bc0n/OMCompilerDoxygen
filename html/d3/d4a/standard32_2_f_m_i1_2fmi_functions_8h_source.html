4 /* This header file must be utilized when compiling a FMU.
5  It defines all functions of Co-Simulation Interface.
6  In order to have unique function names even if several FMUs
7  are compiled together (e.g. for embedded systems), every "real" function name
8  is constructed by prepending the function name by
9  "MODEL_IDENTIFIER" + "_" where "MODEL_IDENTIFIER" is the short name
10  of the model used as the name of the zip-file where the model is stored.
11  Therefore, the typical usage is:
12 
13  #define MODEL_IDENTIFIER MyModel
14  #include "fmiFunctions.h"
15 
16  As a result, a function that is defined as "fmiGetDerivatives" in this header file,
17  is actually getting the name "MyModel_fmiGetDerivatives".
