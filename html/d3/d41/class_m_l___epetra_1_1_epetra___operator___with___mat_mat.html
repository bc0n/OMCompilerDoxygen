Detailed Description

The Epetra_Operator_With_MatMat class is a pure virtual class (specifies interface only) that enable the use of real-valued double-precision operators. It's only addition to the Epetra_Operator is that it adds matmat operations with Epetra_CrsMatrices, returning either an EpetraCrsMatrix or an ML_Operator.
