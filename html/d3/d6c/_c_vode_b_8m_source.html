1 function [varargout] = CVodeB(tout,itask)
2 %CVodeB integrates all backwards ODEs currently defined.
3 %
4 % Usage: [STATUS, T, YB] = CVodeB ( TOUT, ITASK )
5 % [STATUS, T, YB, YQB] = CVodeB ( TOUT, ITASK )
6 %
7 % If ITASK is 'Normal', then the solver integrates from its current internal
8 % T value to a point at or beyond TOUT, then interpolates to T = TOUT and returns
9 % YB(TOUT). If ITASK is 'OneStep', then the solver takes one internal time step
10 % and returns in YB the solution at the new internal time. In this case, TOUT
11 % is used only during the first call to CVodeB to determine the direction of
12 % integration and the rough scale of the problem. In either case, the time
13 % reached by the solver is returned in T.
14 %
15 % If quadratures were computed (see CVodeQuadInitB), CVodeB will return their
16 % values at T in the vector YQB.
17 %
18 % In ITASK =' Normal' mode, to obtain solutions at specific times T0,T1,...,TFINAL
19 % (all increasing or all decreasing) use TOUT = [T0 T1 ... TFINAL]. In this case
20 % the output arguments YB and YQB are matrices, each column representing the solution
21 % vector at the corresponding time returned in the vector T.
22 %
23 % If more than one backward problem was defined, the return arguments are cell
24 % arrays, with T{IDXB}, YB{IDXB}, and YQB{IDXB} corresponding to the backward
25 % problem with index IDXB (as returned by CVodeInitB).
26 %
27 % On return, STATUS is one of the following:
28 % 0: successful CVodeB return.
29 % 1: CVodeB succeded and return at a tstop value (internally set).
30 % -1: an error occurred (see printed message).
31 %
32 % See also CVodeSetOptions, CVodeGetStatsB
