Detailed Description

Base class for boundary control problems with Dirichlet boundary conditions, as formulated by Hans Mittelmann as Examples 1-4 in "Optimization Techniques for Solving Elliptic Control Problems with Control and State Constraints. Part 2: Boundary Control"

Here, the control variables are identical to the values of y on the boundary, and therefore we don't need any explicit optimization variables for u.
