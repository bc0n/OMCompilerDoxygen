Detailed Description

template<class Scalar>
class ModelEvaluator1DPoisson< Scalar >

1D Finite Element model for Poisson's equation

The equation modeled is:

  d2u
  --- = 0
  dx2

  subject to:
     u = a  @ x = 0
     u = b @ x = N

The Matrix W = d(f)/d(x) is implemented as a Thyra::MultiVectorBase object
