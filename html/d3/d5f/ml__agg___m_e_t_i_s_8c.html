Detailed Description

: functions to decompose a local graph using METIS

Note
metis.h is required to properly define indextype, and to declare the function `METIS_PartGraphRecursive' or `METIS_PartGraphKway'. By default, indextype is defined as int, so in principle you can compile also without including metis.h. However, to be sure that use has specified include and library, I require ML_METIS to be defined at compilation time.
Warning
without metis this function compiles, but does not do any job.
