497 /*---------------------------------------------------------------
498  Communication between ARKODE and a ARKODE Linear Solver
499 -----------------------------------------------------------------
500  convfail (input to ark_lsetup)
501 
502  ARK_NO_FAILURES : Either this is the first ark_setup call for
503  this step, or the local error test failed on
504  the previous attempt at this step (but the
505  Newton iteration converged).
506 
507  ARK_FAIL_BAD_J : This value is passed to ark_lsetup if
508 
509  (a) The previous Newton corrector iteration
510  did not converge and the linear solver's
511  setup routine indicated that its Jacobian-
512  related data is not current
513  or
514  (b) During the previous Newton corrector
515  iteration, the linear solver's solve
516  routine failed in a recoverable manner
517  and the linear solver's setup routine
518  indicated that its Jacobian-related data
519  is not current.
520 
521  ARK_FAIL_OTHER : During the current internal step try, the
522  previous Newton iteration failed to converge
523  even though the linear solver was using
524  current Jacobian-related data.
525 ---------------------------------------------------------------*/
526 
527 /* Constants for convfail (input to ark_lsetup) */
528 #define ARK_NO_FAILURES 0
529 #define ARK_FAIL_BAD_J 1
530 #define ARK_FAIL_OTHER 2

564 
565  ypred - the predicted y vector for the current ARKODE internal
566  step.

574  Note: If ark_lsetup calls for re-evaluation of

582 
583  vtemp3 - temporary N_Vector provided for use by ark_lsetup.
