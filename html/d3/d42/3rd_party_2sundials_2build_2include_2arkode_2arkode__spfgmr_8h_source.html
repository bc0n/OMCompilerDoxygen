34 /*---------------------------------------------------------------
35  ARKSpfgmr:
36 
37  A call to ARKSpfgmr links the main ARKODE integrator with the
38  ARKSPFGMR linear solver.
39 
40  arkode_mem is the pointer to the integrator memory returned by
41  ARKodeCreate.
42 
43  pretype is the type of user preconditioning to be done.
44  This must be one of the four enumeration constants
45  PREC_NONE, PREC_LEFT, PREC_RIGHT, or PREC_BOTH defined
46  in sundials_iterative.h.
47  These correspond to no preconditioning,
48  left preconditioning only, right preconditioning
49  only, and both left and right preconditioning,
50  respectively. ARKSPFGMR only implements right
51  preconditioning, so if any one of PREC_LEFT,
52  PREC_RIGHT or PREC_BOTH are defined, it will default
53  to using the right preconditioner only.
54 
55  maxl is the maximum Krylov dimension. This is an
56  optional input to the ARKSPFGMR solver. Pass 0 to
57  use the default value ARKSPFGMR_MAXL=5.
58 
59  The return value of ARKSpfgmr is one of:
60  ARKSPILS_SUCCESS if successful
61  ARKSPILS_MEM_NULL if the arkode memory was NULL
62  ARKSPILS_MEM_FAIL if there was a memory allocation failure
63  ARKSPILS_ILL_INPUT if a required vector operation is missing
64  The above constants are defined in arkode_spils.h
65 
66 ---------------------------------------------------------------*/
67 SUNDIALS_EXPORT int ARKSpfgmr(void *arkode_mem, int pretype, int maxl);
