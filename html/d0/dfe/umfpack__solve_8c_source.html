11 /*
12  User-callable. Solves a linear system using the numerical factorization
13  computed by UMFPACK_numeric. See umfpack_solve.h for more details.
14 
15  For umfpack_*_solve:
16  Dynamic memory usage: UMFPACK_solve calls UMF_malloc twice, for
17  workspace of size c*n*sizeof(double) + n*sizeof(Int), where c is
18  defined below. On return, all of this workspace is free'd via UMF_free.
19 
20  For umfpack_*_wsolve:
21  No dynamic memory usage. Input arrays are used for workspace instead.
22  Pattern is a workspace of size n Integers. The double array W must be
23  at least of size c*n, where c is defined below.
24 
25  If iterative refinement is requested, and Ax=b, A'x=b or A.'x=b is being
26  solved, and the matrix A is not singular, then c is 5 for the real version
27  and 10 for the complex version. Otherwise, c is 1 for the real version and
28  4 for the complex version.
29 */
