Detailed Description

Allgemeine Klasse zur Kapselung der Parameter (Einstellungen) für einen nicht linearen Solver Hier werden default-Einstellungen entsprechend der allgemeinen Simulations- einstellungen gemacht, diese können überprüft und ev. Fehleinstellungen korrigiert werden.
