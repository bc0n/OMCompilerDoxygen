33 " file: ConnectionGraph.mo

35  description: Constant propagation of expressions
36 
37 
38  This module contains a connection breaking algorithm and
39  related data structures. The input of the algorithm is
40  collected to ConnectionGraph record during instantiation.
41  The entry point to the algorithm is findResultGraph.
42 
43  The algorithm is implemented using a disjoint-set
44  data structure that represents the components of
45  elements so far connected.
46  Each component has an unique canonical element.
47  The data structure is implemented by a hash table, that
48  contains an entry for each non-canonical element so that
49  a path beginning from some element eventually ends to the
50  canonical element of the same component.
51 
52  Roots are represented as connections to dummy root
53  element. In this way, all elements will be in the
54  same component after the algorithm finishes assuming
55  that the model is valid.
