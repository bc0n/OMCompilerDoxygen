17 copyright header. A copyright header is determined by looking for beginning
18 @HEADER and ending @HEADER lines. If no such comment blocks are found, then
19 the given copyright header is inserted as the very first line in the file
20 enclosed in '/*' and '*/' lines so as to work also for C files. This can be
