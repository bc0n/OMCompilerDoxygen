30 /********************************
31 * DGESVX uses the LU factorization to compute the solution to a real
32 * system of linear equations
33 * A * X = B,
34 * where A is an N-by-N matrix and X and B are N-by-NRHS matrices.
35 *
36 * Error bounds on the solution and a condition estimate are also
37 * provided.
38 *
39 * Description
40 * ===========
41 *
42 * The following steps are performed:
43 *
44 * 1. If FACT = 'E', real scaling factors are computed to equilibrate
45 * the system:
46 * TRANS = 'N': diag(R)*A*diag(C) *inv(diag(C))*X = diag(R)*B
47 * TRANS = 'T': (diag(R)*A*diag(C))**T *inv(diag(R))*X = diag(C)*B
48 * TRANS = 'C': (diag(R)*A*diag(C))**H *inv(diag(R))*X = diag(C)*B
49 * Whether or not the system will be equilibrated depends on the
50 * scaling of the matrix A, but if equilibration is used, A is
51 * overwritten by diag(R)*A*diag(C) and B by diag(R)*B (if TRANS='N')
52 * or diag(C)*B (if TRANS = 'T' or 'C').
53 *
54 * 2. If FACT = 'N' or 'E', the LU decomposition is used to factor the
55 * matrix A (after equilibration if FACT = 'E') as
56 * A = P * L * U,
57 * where P is a permutation matrix, L is a unit lower triangular
58 * matrix, and U is upper triangular.
59 *
60 * 3. If some U(i,i)=0, so that U is exactly singular, then the routine
61 * returns with INFO = i. Otherwise, the factored form of A is used
62 * to estimate the condition number of the matrix A. If the
63 * reciprocal of the condition number is less than machine precision,
64 * INFO = N+1 is returned as a warning, but the routine still goes on
65 * to solve for X and compute error bounds as described below.
66 *
67 * 4. The system of equations is solved for X using the factored form
68 * of A.
69 *
70 * 5. Iterative refinement is applied to improve the computed solution
71 * matrix and calculate error bounds and backward error estimates
72 * for it.
73 *
74 * 6. If equilibration was used, the matrix X is premultiplied by
75 * diag(C) (if TRANS = 'N') or diag(R) (if TRANS = 'T' or 'C') so
76 * that it solves the original system before equilibration.
77 
78 ********************************/

88 /********************************
89 * DGESVD computes the singular value decomposition (SVD) of a real
90 * M-by-N matrix A, optionally computing the left and/or right singular
91 * vectors. The SVD is written
92 *
93 * A = U * SIGMA * transpose(V)
94 *
95 * where SIGMA is an M-by-N matrix which is zero except for its
96 * min(m,n) diagonal elements, U is an M-by-M orthogonal matrix, and

98 * are the singular values of A; they are real and non-negative, and
99 * are returned in descending order. The first min(m,n) columns of

172 /********************************
173 * DGGEV computes for a pair of N-by-N real nonsymmetric matrices (A,B)
174 * the generalized eigenvalues, and optionally, the left and/or right
175 * generalized eigenvectors.
176 *
177 * A generalized eigenvalue for a pair of matrices (A,B) is a scalar
178 * lambda or a ratio alpha/beta = lambda, such that A - lambda*B is
179 * singular. It is usually represented as the pair (alpha,beta), as
180 * there is a reasonable interpretation for beta=0, and even for both
181 * being zero.
182 *
183 * The right eigenvector v(j) corresponding to the eigenvalue lambda(j)
184 * of (A,B) satisfies
185 *
186 * A * v(j) = lambda(j) * B * v(j).
187 *
188 * The left eigenvector u(j) corresponding to the eigenvalue lambda(j)
189 * of (A,B) satisfies
190 *
191 * u(j)**H * A = lambda(j) * u(j)**H * B .
192 *
193 * where u(j)**H is the conjugate-transpose of u(j).
194 ********************************/
