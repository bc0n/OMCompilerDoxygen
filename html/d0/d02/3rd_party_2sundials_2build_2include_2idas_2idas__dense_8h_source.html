32 /*
33  * -----------------------------------------------------------------
34  * Function : IDADense
35  * -----------------------------------------------------------------
36  * A call to the IDADense function links the main integrator
37  * with the IDADENSE linear solver module.
38  *
39  * ida_mem is the pointer to integrator memory returned by
40  * IDACreate.
41  *
42  * Neq is the problem size
43  *
44  * IDADense returns:
45  * IDADLS_SUCCESS = 0 if successful
46  * IDADLS_LMEM_FAIL = -1 if there was a memory allocation failure
47  * IDADLS_ILL_INPUT = -2 if NVECTOR found incompatible
48  *
49  * NOTE: The dense linear solver assumes a serial implementation
50  * of the NVECTOR package. Therefore, IDADense will first
51  * test for a compatible N_Vector internal representation
52  * by checking that the functions N_VGetArrayPointer and
53  * N_VSetArrayPointer exist.
54  * -----------------------------------------------------------------
55  */
