pair< string, string > OMCFactory::parseIngoredAndWrongFormatOption ( const string &  s)
protected

This helper-function is invoked by the boost program option library and will handle options in the c-runtime format and options that should be ignored. It parses a long option that starts with one dash, like '-port=12345' and put it into the 'unrecognized' category. If an option is detected which is part of the arguments to ignore list, it is put into the 'ignored' category.

Parameters
Theargument that should be handled.
Returns
The pair of category and value that should be used for the given argument.
