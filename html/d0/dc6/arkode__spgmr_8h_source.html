34 /*---------------------------------------------------------------
35  ARKSpgmr:
36 
37  A call to the ARKSpgmr function links the main ARKODE integrator
38  with the ARKSPGMR linear solver.
39 
40  arkode_mem is the pointer to the integrator memory returned by
41  ARKodeCreate.
42 
43  pretype is the type of user preconditioning to be done.
44  This must be one of the four enumeration constants
45  PREC_NONE, PREC_LEFT, PREC_RIGHT, or PREC_BOTH defined
46  in sundials_iterative.h.
47  These correspond to no preconditioning,
48  left preconditioning only, right preconditioning
49  only, and both left and right preconditioning,
50  respectively.
51 
52  maxl is the maximum Krylov dimension. This is an
53  optional input to the ARKSPGMR solver. Pass 0 to
54  use the default value ARKSPGMR_MAXL=5.
55 
56  The return value of ARKSpgmr is one of:
57  ARKSPILS_SUCCESS if successful
58  ARKSPILS_MEM_NULL if the arkode memory was NULL
59  ARKSPILS_MEM_FAIL if there was a memory allocation failure
60  ARKSPILS_ILL_INPUT if a required vector operation is missing
61  The above constants are defined in arkode_spils.h
62 
63 ---------------------------------------------------------------*/
64 SUNDIALS_EXPORT int ARKSpgmr(void *arkode_mem, int pretype, int maxl);
