Detailed Description

template<class SparseMatrixType>
class Belos::Tpetra::HarwellBoeingReader< SparseMatrixType >

Read a Harwell-Boeing format file into a Tpetra::CrsMatrix.

Author
Mark Hoemmen
Warning
This reader is incomplete. It currently only handles real double-precision floating-point data and indices of Fortran type integer, and only some of the possible ways that these can be represented in the file.
