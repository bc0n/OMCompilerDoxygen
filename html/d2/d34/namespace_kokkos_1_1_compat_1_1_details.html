bool Kokkos::Compat::Details::getVerboseParameter ( const Teuchos::ParameterList &  params)

Get the value of the "Verbose" parameter as a bool.

This method lets the "Verbose" parameter have type either int or bool, and returns its value as bool. If the "Verbose" parameter does not exist in the given list, return the default value, which is false.
