1 FMI Library
2 ===========
3 
4 FMI Library (FMIL) is a software package written in C that enables integration
5 of Functional Mock-up Units (FMUs) import in applications. FMI Library is an
6 independent open-source implementation of the FMI open standard
7 (www.fmi-standard.org).
8 More information can be found at the FMI Library website www.fmi-library.org.
9 
10 The FMI Library code is hosted at https://svn.jmodelica.org/FMILibrary/.
11 There is a git mirror at https://github.com/modelon/FMILibrary.
12 Active issues are tracked at http://trac.jmodelica.org/wiki/FMILibrary.
13 Discussion forums for FMI Library can be found at http://jmodelica.org/forum.
14 Please report issues related to FMI Library at the forums.
