1 %CVPrecSolveFn - type for user provided preconditioner solve function.
2 %
3 % The user-supplied preconditioner solve function PSOLFN
4 % is to solve a linear system P z = r in which the matrix P is
5 % one of the preconditioner matrices P1 or P2, depending on the
6 % type of preconditioning chosen.
7 %
8 % The function PSOLFUN must be defined as
9 % FUNCTION [Z, FLAG] = PSOLFUN(T,Y,FY,R)
10 % and must return a vector Z containing the solution of Pz=r.
11 % If PSOLFUN was successful, it must return FLAG=0. For a recoverable
12 % error (in which case the step will be retried) it must set FLAG to a
13 % positive value. If an unrecoverable error occurs, it must set FLAG
14 % to a negative value, in which case the integration will be halted.
15 % The input argument FY contains the current value of f(t,y).
16 %
17 % If a user data structure DATA was specified in CVodeInit, then
18 % PSOLFUN must be defined as
19 % FUNCTION [Z, FLAG, NEW_DATA] = PSOLFUN(T,Y,FY,R,DATA)
20 % If the local modifications to the user data structure are needed in
21 % other user-provided functions then, besides setting the vector Z and
22 % the flag FLAG, the PSOLFUN function must also set NEW_DATA. Otherwise,
23 % it should set NEW_DATA=[] (do not set NEW_DATA = DATA as it would
24 % lead to unnecessary copying).
25 %
26 % See also CVPrecSetupFn, CVodeSetOptions
27 %
28 % NOTE: PSOLFUN is specified through the property PrecSolveFn to
29 % CVodeSetOptions and is used only if the property LinearSolver was
30 % set to 'GMRES', 'BiCGStab', or 'TFQMR' and if the property PrecType
31 % is not 'None'.
